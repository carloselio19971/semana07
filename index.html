Definicion RWD

Responsive Web Desing nos permite que nuestra pagina web, aplicación sea adaptable a dispositivos moviles como tables, mobiles , etc, basado en un conjuntos de practicas para lograrlo

Enfoques de Responsive Web Desing

Es importante tener encuenta que los enfoques deben estar en coordinacion con el Desing UX/UI

Existes 2 enfoque dentro del Responsive Web Desing Desktop First y Mobile First

Desktop First

Este enfoque nos recomienda iniciar el desarrollo desde un version de escritorio, hasta una version mobil, iniciando el proyecto desde tamñano Desktop hasta llegar al tamaño mobile

Es recomendable usar este enfoque si es que tenemos experiencia en desarrollo, ya que tendremos que ocultar algunos elementos para adaparlo a dispositos mas pequeños.Se utiliza la propiedad css (max-width)

Ejemplo

Si necesitamos realizar una diseño web para una aplicacion de farmacia y nuestro diseñador ya tiene el diseño de Desktop listo , deberiamos utilizar esta metodologia.

Mobile First

Este enfoque nos recomienda iniciar el desarrollo desde una version movil hasta una version de escritorio, es ideal para progamadores junior que recien inician , ayuda a optimizar tiempos debido a que no tendremos que ocultar elementos como en la metdologia Desktop First

Ejemplo

Si queremos desarorllar una pagina web para ventas , y aun nuestro diseñador no tiene un diseño para escritorio y nosotros como desarrolladores tenemos poca experiencia, y tenemos poco tiempo para hacer el proyecto se recomienda usar este enfoque, construyendo primero el diseño para mobiles y luego ir adaptando y agregando nuevos elementos hasta que el Desktop Desing sea realizado por UX/UI

Media Query

Son puntos especificos en donde nosotros podemos establecer limites a los distitos estilos para el mismo elemento, se utilizar 2 propiedades @media para definir los limites o Break Points

Ejemplo

Se nos indica que tenemos que tener un circulo que para un ancho maximo 600 px sea negro , luego cuando el ancho maximo sea 500 px sea rojo y finalmente cuando el ancho maximo sea 400px se verde

Media Query (Max-widh:600) Negro

Media Query (Max-widh:500) Rojo

Media Query (Max-widh:400) Verde

Ejemplo Practico Media Query enfoque Destop First

Ejemplo

Tengo 4 cajas posicionadas con flex box para mi pagina web en Desktop y ahora quiero que cuando se encuentre en dispositivos mobiles , las cajas se ponene una sola fila cuando el ancho sea menor que 500px

Cuando el elemento caja llega a un Max-widh:500 se cambiar el eje principal a columnas con la propiedad flex-direction: column;

Cuando el footer se menor de 600px de ancho se alinea en una sola fila
